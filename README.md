# ⬡ OBJECT MEASURE — 물체 측정기

스마트폰 카메라로 물체를 비추면 실시간으로 **물체 인식**, **치수 측정(mm)**, **중량 추정(kg)**을 수행하는 웹 애플리케이션입니다.

---

## 주요 기능

### 실시간 물체 감지
- TensorFlow.js + COCO-SSD 모델 기반
- 80종류 물체 자동 인식 (사람, 자동차, 노트북, 컵, 의자 등)
- 초록색 바운딩 박스 + 코너 강조 오버레이
- 인식 신뢰도(%) 표시

### 치수 측정
- 가로(W), 세로(H) mm 단위 표시
- 시안색 점선 치수선 화면 오버레이
- 핀홀 카메라 모델 기반 크기 추정
- 거리 슬라이더(20~300cm)로 보정 가능

### 중량 추정
- 80종 물체별 실제 평균 중량 데이터베이스 내장
- 소수점 2자리 kg 단위 (예: `0.19 kg`)
- 바운딩 박스 크기 비율에 따른 동적 보정

### UI / UX
- HUD 스타일 다크 테마 (네온 그린 + 시안 액센트)
- Orbitron / Rajdhani 타이포그래피
- 스캔라인 효과 + 코너 브래킷 프레임
- 하단 정보 패널에 감지 결과 카드 표시
- 모바일 최적화 (터치, 반응형)

## 실행 방법

### 1. 로컬 실행 (HTTPS 필수)

카메라 API는 보안 컨텍스트(HTTPS 또는 localhost)에서만 동작합니다.

```bash
# Python 간이 서버 (localhost)
python3 -m http.server 8000

# 브라우저에서 접속
# http://localhost:8000/object-measure.html
```

### 2. HTTPS 환경

외부 기기에서 접속하려면 HTTPS가 필요합니다.

```bash
# mkcert로 로컬 인증서 생성
mkcert -install
mkcert localhost 127.0.0.1

# Node.js 간이 HTTPS 서버 또는 ngrok 사용
npx serve --ssl-cert ./localhost.pem --ssl-key ./localhost-key.pem
```

### 3. 모바일 접속

1. PC와 같은 Wi-Fi에 연결
2. HTTPS 서버 주소로 접속
3. 카메라 권한 허용
4. 물체를 비추면 자동 측정 시작

---

## 사용 방법

1. **카메라 허용** — 최초 실행 시 카메라 접근 권한을 허용합니다.
2. **모델 로딩 대기** — COCO-SSD 모델이 로딩될 때까지 잠시 기다립니다 (최초 3~10초).
3. **물체 비추기** — 후면 카메라로 측정할 물체를 비춥니다.
4. **거리 조절** — 하단 슬라이더로 카메라~물체 거리(cm)를 맞추면 정확도가 올라갑니다.
5. **결과 확인** — 화면 오버레이와 하단 패널에서 치수·중량을 확인합니다.

## 인식 가능 물체 (80종)

일상 용품, 가전, 가구, 음식, 동물, 탈것, 스포츠 용품 등 COCO 데이터셋 기반 80개 카테고리를 지원합니다.

**예시:** 사람, 자전거, 자동차, 오토바이, 고양이, 개, 배낭, 우산, 병, 컵, 포크, 칼, 바나나, 사과, 피자, 의자, 소파, TV, 노트북, 마우스, 키보드, 스마트폰, 냉장고, 책, 시계, 가위, 칫솔 등

## 측정 정확도 안내

> ⚠️ 본 앱의 치수 및 중량은 **추정값**입니다.

- 치수는 핀홀 카메라 모델 근사와 물체별 평균 크기 데이터를 조합하여 계산합니다.
- 중량은 물체 카테고리별 평균 무게를 기반으로 바운딩 박스 비율에 따라 보정합니다.
- 거리 슬라이더를 실제 거리에 맞출수록 정확도가 높아집니다.
- 정밀 측정이 필요한 경우 전용 측정 장비를 사용하세요.
